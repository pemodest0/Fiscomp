# Fiscomp

Computational physics materials for teaching numerical simulation in an introductory and didactic format.

## Current structure
- `pendulo_simples/`
  - `pendulo_simples.py`
  - `pendulo_simples.ipynb`
- `massa_mola/`
  - `massa_mola.py`
  - `massa_mola.ipynb`

## Each model includes
1. Euler simulation
2. Runge-Kutta 4 (RK4) simulation
3. Raw numerical output (initial table)
4. Comparative plots

## Notes
- The `.py` files are intentionally documented in PT-BR.
- Notebooks and repository documentation are in English.
